# AI Voice Agent Live


## Overview
This project showcases a live multilingual AI voice assistant integrated into a real website environment.

Users can interact with the assistant directly through voice or text in multiple languages.

## Live Demo
Available on AddVANCEO:
https://addvanceo.eu

## What Users Can Do
- Start a live conversation directly from the website
- Interact through voice and text
- Switch between multiple languages
- Experience an AI-powered support flow in a real interface

## Highlights
- Multilingual interaction
- Website-integrated assistant
- Real user-facing implementation
- Clean and intuitive interface

## Notes
This implementation is part of a live environment, so the underlying code is not publicly shared.
